# PROLANG Final Project

## JavaScript/TypeScript vs Python — Language Implementation Audit

## Project Overview

This project is a comparative programming language audit between **JavaScript/TypeScript** and **Python**. The notebook demonstrates how both languages implement core programming concepts, including syntax, performance, memory efficiency, readability, and maintainability.

The project was created as part of the **Programming Languages Final Project Requirement**.

---

# Objectives

* Compare implementation styles between JavaScript/TypeScript and Python
* Demonstrate core programming concepts using working examples
* Analyze language performance and memory usage
* Identify code smells and apply refactoring techniques
* Reflect on language design and developer experience

---

# Included Contents

The project notebook contains:

* Data Types & Variables
* Functions & Scope
* Conditionals & Loops
* Collections & Data Structures
* Object-Oriented Programming
* Error & Exception Handling
* Asynchronous Programming
* Performance & Memory Analysis
* Code Smell Detection & Refactoring
* Comparative Reflections

Each concept includes multiple functional examples for verification and analysis.
